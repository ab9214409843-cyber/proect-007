# Этап 5. Автоматический план подготовки

## Цель
При создании мероприятия система сама создаёт список задач по дате проведения (за 90/60/30/7/1 день).

## Задача для Claude (новое окно)
> Добавь автоматическую генерацию задач при создании мероприятия на основе даты проведения.
> Шаблоны задач берём из таблицы `checklist_templates` (поле `offset_days` — за сколько дней до даты).

## Важно: используем реальную методичку ФЦ БАС
Шаблон задач строим не на абстрактном примере, а на **реальных этапах из методички**:
`C:\Claude_Code\methodology\methodology.md` (9-этапный процесс организации мероприятий).
Так автоплан станет рабочим инструментом, а не демонстрацией.

## Пример шаблона (из дорожной карты — заменим/дополним по методичке)
- За 90 дней: определить концепцию; выбрать площадку.
- За 60 дней: подготовить положение; согласовать подрядчиков.
- За 30 дней: подготовить сценарий; заказать наградную продукцию.
- За 7 дней: проверить логистику; провести финальную сверку.
- За 1 день: проверить площадку; подтвердить готовность команды.

## Чек-лист этапа
- [ ] Таблица `checklist_templates` наполнена (на базе методички ФЦ БАС).
- [ ] При создании мероприятия задачи создаются автоматически.
- [ ] У каждой задачи правильно посчитан срок (`due_date = дата мероприятия − offset_days`).
- [ ] Созданные задачи видны в карточке мероприятия и в `/tasks`.

## Результат этапа
Пользователь создаёт мероприятие → система сама создаёт план задач с корректными сроками.
