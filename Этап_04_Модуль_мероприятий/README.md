# Этап 4. Модуль мероприятий

## Цель
Сделать полный CRUD по мероприятиям: список, создание, карточка, редактирование, удаление, смена статуса.

## Задача для Claude (новое окно)
> Сделай CRUD для мероприятий: создание, список, редактирование, удаление, карточка мероприятия.

## Чек-лист этапа

Список мероприятий (`/events`):
- [ ] Посмотреть все мероприятия.
- [ ] Создать мероприятие.
- [ ] Открыть карточку мероприятия.
- [ ] Изменить статус мероприятия.

Карточка мероприятия (`/events/[id]`) показывает:
- [ ] название;
- [ ] дату;
- [ ] место;
- [ ] статус;
- [ ] задачи (привязанные);
- [ ] документы (привязанные);
- [ ] заметки опыта (привязанные).

Статусы мероприятия:
- планируется
- в подготовке
- проводится
- завершено

## Результат этапа
Мероприятия можно создавать, смотреть списком, открывать карточку, менять статус, редактировать и удалять.
