# Этап 4. Модуль мероприятий

## Цель
Сделать полный CRUD по мероприятиям: список, создание, карточка, редактирование, удаление, смена статуса.

## Задача для Claude (новое окно)
> Сделай CRUD для мероприятий: создание, список, редактирование, удаление, карточка мероприятия.

## Чек-лист этапа

Список мероприятий (`/events`):
- [x] Посмотреть все мероприятия.
- [x] Создать мероприятие.
- [x] Открыть карточку мероприятия.
- [x] Изменить статус мероприятия.

Карточка мероприятия (`/events/[id]`) показывает:
- [x] название;
- [x] дату;
- [x] место;
- [x] статус;
- [~] задачи (привязанные) — блок-заглушка, наполнится на этапе 5;
- [~] документы (привязанные) — блок-заглушка, наполнится на этапе 6;
- [~] заметки опыта (привязанные) — блок-заглушка, наполнится на этапе 7.

Статусы мероприятия:
- планируется
- в подготовке
- проводится
- завершено

## Результат этапа
Мероприятия можно создавать, смотреть списком, открывать карточку, менять статус, редактировать и удалять.

## Статус
`завершён ✅`

## Заметки
- Код: `app/src/app/(cabinet)/events/` (список, `new`, `[id]`, `[id]/edit`, `actions.ts`,
