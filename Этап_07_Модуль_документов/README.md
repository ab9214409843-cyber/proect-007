# Этап 7. Модуль документов

## Цель
Библиотека документов: загрузка файлов, привязка к мероприятию, тип документа, скачивание, удаление.
Плюс библиотека шаблонов документов (пока без генератора).

## Задача для Claude (новое окно)
> Сделай модуль документов: загрузка файлов (Supabase Storage), список документов, тип документа,
> связь с мероприятием. Добавь раздел «Шаблоны документов» как библиотеку (список + описание +
> кнопка скачать/использовать).

## Чек-лист этапа

Документы (`/documents`):
- [ ] Загрузить документ (в Supabase Storage).
- [ ] Привязать документ к мероприятию.
- [ ] Указать тип документа.
- [ ] Скачать документ.
- [ ] Удалить документ.

Типы документов:
- положение, регламент, служебная записка, смета, ТЗ, сценарный план, отчёт, другое.

Шаблоны документов (задача 10):
- [ ] Список шаблонов.
- [ ] Описание шаблона.
- [ ] Кнопка скачать/использовать.
- [ ] (Позже) автозаполнение данными мероприятия.

> Файлы шаблонов лежат в папке `_шаблоны/` — см. [../_шаблоны/README.md](../_шаблоны/README.md).

## Результат этапа
Документы можно загружать, привязывать к мероприятию, скачивать и удалять; есть библиотека шаблонов.
