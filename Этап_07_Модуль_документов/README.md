## Заметки
- **Хранилище:** приватный бакет `documents` (Supabase Storage), лимит 25 МБ; RLS на
  `storage.objects` — пользователь работает только со своей папкой `{user_id}/...`. Скачивание —
  через временную подписанную ссылку (бакет приватный, прямых ссылок нет). Миграция —
  `app/supabase/migrations/0006_documents_storage.sql` (+колонки `file_name`, `file_size`).
- **Код:** список/фильтры `/documents`, загрузка `/documents/new`, карточка `/documents/[id]`,
  редактирование метаданных `/documents/[id]/edit`, удаление с подтверждением, библиотека шаблонов
  `/documents/templates`. Логика — server actions `documents/actions.ts`; общая форма
  `DocumentForm.tsx`; хелперы/типы/шаблоны — `app/src/lib/documents.ts`. В карточке мероприятия —
  секция «Документы» + «+ Добавить документ».
- **Конфиг:** в `next.config.ts` поднят `serverActions.bodySizeLimit` до 25 МБ (по умолчанию 1 МБ —
  загрузка падала бы).
- **Каркас:** замена самого файла при редактировании не делается (только метаданные); кнопка у
  шаблонов — заглушка, реальные файлы и автозаполнение — позже.
