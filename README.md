# Proect_007

Веб-приложение для организаторов спортивных мероприятий: создать мероприятие → получить
автоматический план подготовки → вести задачи → хранить документы → фиксировать опыт.

## С чего начать

1. Прочитать [CLAUDE.md](CLAUDE.md) — главный контекст и текущий статус.
2. Посмотреть дорожную карту: [docs/roadmap.md](docs/roadmap.md).
3. Открыть папку текущего этапа (начинаем с `Этап_01_Подготовка_проекта/`) и работать по её README.

## Что где лежит

- **`CLAUDE.md`** — контекст проекта и прогресс (читается Claude автоматически).
- **`docs/`** — дорожная карта, стек, черновик схемы базы данных.
- **`app/`** — само приложение (Next.js). Появится на этапе написания кода.
- **`_шаблоны/`** — библиотека шаблонов документов.
- **`Этап_01 … Этап_11`** — по папке на каждый этап: задачи, чек-лист, статус.

## Принцип работы

Новое окно Claude Code — на каждую задачу. Открываешь папку этапа, читаешь README, идёшь по чек-листу,
отмечаешь статус. Код пишем только в `app/`.
