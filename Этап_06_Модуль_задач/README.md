# Этап 6. Модуль задач

## Цель
Полное управление задачами: создание, изменение, удаление, смена статуса, привязка к мероприятию.

## Задача для Claude (новое окно)
> Сделай модуль задач: список задач, статусы, дедлайны, приоритеты, связь с мероприятием.

## Чек-лист этапа

Функции:
- [ ] Создать задачу.
- [ ] Изменить задачу.
- [ ] Удалить задачу.
- [ ] Поменять статус.
- [ ] Привязать задачу к мероприятию.

Страница `/tasks` — список всех задач (с фильтрами по статусу/мероприятию).

Карточка задачи показывает:
- [ ] название;
- [ ] описание;
- [ ] срок;
- [ ] статус;
- [ ] приоритет;
- [ ] мероприятие.

Статусы задачи:
- к выполнению
- в работе
- на согласовании
- выполнено

## Результат этапа
Задачами можно полноценно управлять, они связаны с мероприятиями, видны списком и в карточке мероприятия.
