# Этап 2. Структура продукта

## Цель
Сделать публичную главную страницу и страницы входа/регистрации, а также каркас личного кабинета
(все основные разделы как страницы).

## Задача для Claude (новое окно)
> Сделай публичную главную страницу с описанием продукта, проблемой организаторов, основными
> возможностями, блоком MVP и кнопкой регистрации/входа. Добавь страницы входа и регистрации.
> Сделай каркас личного кабинета со всеми разделами.

## Чек-лист этапа

Публичная часть:
- [x] `/` — главная (описание продукта, проблема, как это работает, возможности, для кого, призыв к действию, кнопки вход/регистрация).
- [x] `/login` — вход (с этапа 1).
- [x] `/register` — регистрация (с этапа 1).

Личный кабинет (каркас страниц, наполнение — на следующих этапах):
- [x] `/dashboard` — обзор.
- [x] `/events` — мероприятия.
- [x] `/events/[id]` — карточка мероприятия.
- [x] `/tasks` — все задачи.
- [x] `/documents` — документы.
- [x] `/experience` — база опыта.

## Результат этапа
Есть публичная страница, страницы входа/регистрации и все разделы кабинета (пока пустые каркасы),
переходы между ними работают.
